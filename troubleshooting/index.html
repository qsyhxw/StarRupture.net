Safe-First Rules (Read Before You Nuke Your Settings)

Troubleshooting is fastest when you reduce risk and isolate variables.

Do this order

  • Start with reversible steps: verify files, restart, remove overlays, test once.
  • Change one thing at a time, so you know what actually fixed it.
  • Avoid sketchy “fix packs” and random executables.
  • When in doubt: report with clean repro steps (we’ll turn it into a page).

Report Template (Copy/Paste)

If you email us a clean report, we can verify + publish a short fix page faster.

Minimal template

  • Title: [Crash / Freeze / Black Screen / Controls] + when it happens
  • Platform: PC/Steam
  • When: first launch / loading / joining co-op / after changing settings
  • Repro: 1) … 2) … 3) …
  • Expected vs Actual: …
  • PC Specs: CPU / GPU / RAM / OS / driver version (if known)
  • What you tried: verified files? closed overlays? restarted?
